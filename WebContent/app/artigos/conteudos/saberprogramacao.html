O que é HTML?

HTML é a abreviatura de “HyperText Mark-up Language” ou Linguagem de Marcação de Hipertexto, em português. Usamos o HTML para construir e apresentar o conteúdo no navegador através principalmente de tags( <head>, <title>, <body>, etc…).

Muitos designers e desenvolvedores não tratam o HTML como uma linguagem de programação, pois o mesmo não apresenta uma lógica programação (como o JavaScript por exemplo).

O HTML não envolve a necessidade de um profundo pensamento matemático. Ele nada mais é do que uma estrutura, em código, dos elementos da página (como em um quebra cabeça). Algo que, de certa forma, pode ser comparado ao processo de diagramação, mas em código.

O que acha de criar seu primeiro HTML? Siga as instruções.

- Abra um Bloco de Notas (Windows) ou um TextEdit (Mac). - Copie e cole o texto abaixo. - Salve o arquivo com o nome index.html e selecione Todos os Arquivos no campo Tipo. - Feche o arquivo e abra-o novamente. - Tcharãã =]... você acaba de criar seu primeiro arquivo HTML (fácil não?!)

<!DOCTYPE html>
<html lang="pt-br">
<head>
<meta charset="utf-8">
<title>Designer do Futuro</title>
</head>
<body>
<h1>Designer do Futuro</h1>
<p>Quer estar preparado quando aquela oportunidade de trabalho chegar? Deixa a gente ajudar! Nós da Design do Futuro estamos desenvolvendo, com a ajuda de alguns parceiros um curso que fará com que você se <b>destaque no mercado de trabalho</b>. E sabe o melhor? O curso será <b>custeado quase que inteiramente por empresas parceiras, que já estarão esperando você do lado de fora</b>. Mas as vagas serão limitadas. Não vai deixar essa oportunidade passar hein! <b>Pra ser o primeiro a saber cadastre seu nome completo e e-mail aqui!</b></p>
</body>
</html>

O que é CSS?

CSS é a abreviatura de Cascading Style Sheets ou Folhas de Estilos que são compostas por “camadas”. Trabalha em conjunto com as linguagens XML, XHTML e, principalmente, HTML. Sua função é ler e apresentar essas linguagens em páginas da internet, dando-às aparência.

O CSS define como serão exibidos os elementos contidos no código de uma página da internet e sua maior vantagem é efetuar a separação entre o formato e o conteúdo de um documento. Encarregando-se da aplicação dos estilos e formatos necessários.

O CSS também permite que as mesmas marcações de uma página sejam apresentadas em diferentes estilos, conforme os métodos de renderização (como em uma tela, impressão, via voz, baseadas em dispositivos touchs, etc.)

O que é JavaScript?

Primeiro de tudo, é importante esclarecer que JavaScript e Java são linguagens completamente diferentes, e que uma independe da outra. Inicialmente o JavaScript se chamava LiveScript, mas, na época, por conta de uma estratégia mercadológica, seu nome foi alterado em uma tentativa de aproveitar a onda de sucesso da linguagem Java, desenvolvida pela Sun Microsystems. Lembre-se, JavaScript e Java não se tratam da mesma linguagem.

Agora sim, vamos ao que realmente interessa a você querido designer. O que é JavaScript? Se trata de uma linguagem de programação que permite ao usuário implementar itens complexos a páginas web. Ou seja, qualquer página que apresenta informações não estáticas, provavelmente está usando JavaScript. Como exemplo temos mapas interativos, sub-menus flutuantes, conteúdos que se atualizam em certo intervalo de tempo, imagens animadas, entre outros. Basicamente, tudo que é dinâmico.

Uma das principais características do JavaScript é que ele roda programas localmente (do lado do cliente), o chamado client-side. Assim sendo, fornece às páginas web a possibilidade de programação, transformação e processamento de dados enviados e recebidos, interagindo com a marcação e exibição de conteúdo da linguagem HTML e com a estilização desse conteúdo proporcionada pelo CSS nessas páginas.

Outra característica muito importante do JavaScript é que ele é compatível com todos os browsers da atualidade, podendo por isso poupar uma terrível dor de cabeça quando o assunto é responsividade. Trazendo os elementos certos, para os usuários corretos.

Em resumo, o JavaScript adiciona comportamento interativo à página, modificando assim sua utilização e proporcionando diferentes experiências ao utilizador.

Agora, vamos desenvolver seu primeiro código JavaScript?

Para isso, siga as mesmas instruções localizadas na parte do HTML, dessa vez colando o texto abaixo no documento criado.

<html>
<body>
<h1>
<u>Designer do Futuro</u>
</h1>
<hr>
<p id="demo">javaScript can change HTML content.</p>
<button type="button" onclick="document .getElementById('demo') .innerHTML = 'Qualificando profissionais para o mercado de trabalho. Vem com a gente! =]'">Clique aqui!</button>
</body>
</html>

Resumindo tudo, é imprescindível ter noções básicas das línguas HTML, CSS e JavaScript para qualquer designer que queira se destacar no mercado de trabalho, pois essas três linguagens juntas compõem a maior parte da programação que é ligada ao trabalho de um designer.

Para não esquecer, lembre sempre que o HTML é responsável pelo conteúdo, o CSS pelo formato e o JavaScript pela interatividade de uma página web.
